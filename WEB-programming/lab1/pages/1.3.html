ЗАДАНИЕ 1.3

Найти в интернете описание работы DNS, узнать что такое: ns-сервер, делегирование домена, A-запись, MX-запись.

Работа DNS

DNS (Domain Name System) - это система, которая переводит доменные имена в IP-адреса и наоборот. Это делается для удобства пользователей, так как доменные имена легче запомнить и использовать, чем IP-адреса. DNS серверы расположены по всему миру и обмениваются информацией между собой для обеспечения корректной работы интернета.

NS-сервер

NS-сервер (Name Server) - это сервер, который хранит информацию о доменных именах и соответствующих им IP-адресах. Когда пользователь вводит доменное имя в браузер, запрос отправляется на NS-сервер, который возвращает соответствующий IP-адрес. Затем браузер использует этот IP-адрес для доступа к веб-сайту.

Делегирование домена

Делегирование домена - это процесс передачи ответственности за доменное имя от одного NS-сервера другому. Это может быть необходимо, например, при переходе на другой хостинг или при изменении DNS-провайдера. В результате делегирования домена, новый NS-сервер становится ответственным за разрешение доменного имени и предоставление соответствующих IP-адресов.

A-запись

A-запись (Address Record) - это запись в DNS, которая указывает на IP-адрес, соответствующий доменному имени. A-записи используются для того, чтобы направлять трафик с доменного имени на определенный IP-адрес сервера, на котором размещен веб-сайт.

MX-запись

MX-запись (Mail Exchange Record) - это запись в DNS, которая указывает на сервер, ответственный за обработку электронной почты, отправленной на адреса, связанные с доменным именем. MX-записи используются для маршрутизации электронной почты и определения сервера, который принимает и обрабатывает входящие сообщения для данного домена.
